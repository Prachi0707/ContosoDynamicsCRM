# ContosoDynamicsCRM
Shows unit testing of plugins for a sample plugin implemented for the fictitious Contoso Organization

## Pre-requisites ##
1. Dynamics 365 project templates (package, plugin) and Visual Studio 2015

## How to build and other details ##
1. Please rebuild the solution to resolve all the Nuget package references
2. The Dynamics 365 SDK, project template and assemblies have been used to develop the plugin
3. The master branch contains the implementation of the plugin and all the unit test methods
4. Code coverage for the developed plugin is > 80% with the developed unit tests
5. All Fakes references have been added to source control
6. The Fakes references for the System assemblies need to have the specific version that is used.
